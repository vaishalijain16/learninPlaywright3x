# VS Code Shortcuts — macOS

## General

| Command                          | Shortcut               |
|-----------------------------------|-------------------------|
| Command Palette                   | `Cmd+Shift+P`           |
| Quick Open (go to file)           | `Cmd+P`                 |
| New Window                        | `Cmd+Shift+N`           |
| Close Window                      | `Cmd+Shift+W`           |
| Settings                          | `Cmd+,`                 |
| Keyboard Shortcuts                | `Cmd+K Cmd+S`           |
| Toggle Full Screen                | `Ctrl+Cmd+F`            |
| Zoom In / Out                     | `Cmd+=` / `Cmd+-`       |

## Editing

| Command                          | Shortcut               |
|-----------------------------------|-------------------------|
| Cut / Copy / Paste line (no selection) | `Cmd+X` / `Cmd+C` / `Cmd+V` |
| Move Line Up / Down               | `Option+↑` / `Option+↓` |
| Copy Line Up / Down               | `Shift+Option+↑` / `Shift+Option+↓` |
| Delete Line                       | `Cmd+Shift+K`           |
| Insert Line Below / Above         | `Cmd+Enter` / `Cmd+Shift+Enter` |
| Undo / Redo                       | `Cmd+Z` / `Cmd+Shift+Z` |
| Toggle Line Comment                | `Cmd+/`                 |
| Toggle Block Comment               | `Shift+Option+A`        |
| Indent / Outdent Line             | `Cmd+]` / `Cmd+[`       |
| Format Document                   | `Shift+Option+F`        |
| Rename Symbol                     | `F2`                     |
| Trigger Suggest (autocomplete)    | `Ctrl+Space`             |
| Quick Fix                         | `Cmd+.`                 |
